# bowling-console-app
A bowling console application for one player

Requires Java 1.8 or greater

Main menu:

```
| X | - |    |    |    |    |    |    |    |    |
1.-     Add Roll
2.-     Get current Frame
3.-     Get current Roll
4.-     Get current Score
5.-     Restart
6.-     Quit
ENTER.- Print this menu    
```

Steps for running:

In Windows/Linux
```
gradlew run
```
